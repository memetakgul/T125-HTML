CSS: Cascading Style Sheets

(merhaba)

Cascading: parent elemente uygulanan style islemi child larar da etki etmesini saglar.
Bir elementin ecerisinde baska bir element olusturuldugunda disaridaki parent icerideki child olur.

inline CSS: elementlerin acilis tag i icersinde style attribute ile css isleminin yapilmasidir.
Childe yapilan CSS style, parent a yapilan css style a override eder.
