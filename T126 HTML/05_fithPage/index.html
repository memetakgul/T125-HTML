CSS in bazi ozellikleri

  • color: texting rengini belirler.
  • font-family: yazinin tipini belirler.
    • times new roman
    • Ariel
    • Verdana
  • Font size: yazi buyuklugunu belirler.
  • Border: html elementinin etrafina border sinir cizer.
  • margin: border in disina dort bir tarafindan bosluk birakir.
  • padding: border ile metin arasinda bosluk birakir.
  • selector: id, class, tag vb. olusturacagi bir arada kullanim kombinasyonlarina denir.
    • #idName, .className, tagName{ hepsi icin ayni style vb islemi yapabiliriz. }
    #selectorExample, .selectorExample2, p { color: rgb(22, 121, 40); }
