Attribute

Bir elementte birden fazla attribute kullanilabilir. Attributeler arsinda bosluk birakarak veya ; ile ayirarak birden fazla eklenilmasi saglanabilir.


height="35%" name=value pair lari seklinde kullanilir. attribute daima acilis tagine eklenir

id Attribute

  • HTML element leri icin unique id olusturulmasini saglar.
  • CSS ve JavaScript, belirli bir elementle ilgili (style/select)
    • CSS te belirli bir elemente #ElementId ile ulasilabilir.
    • JavaScript te belirli bir elemente getElementByID() ile ulasilabilir
    • id case sensitivedir. Buyuk kucuk harf fark eder.

Class Attribute

Elementler icin Class (container) olusturur. CSS ve JavaScript ten ayni class attribute e sahip class name e sahip elementlere ulasmayi ve ilgili islem yapmayi saglar.

  • Bir element icin birden fazla class belirlenebilir.
  • Class attribute Global attribute tur. Neredeyse butun elementlerde kullanilabilir.
  • Farkli elementler ayni class attribute une sahip olabilir.
  • CSS ten bir class a ulasmak icin " .ClassName " ile ulasilir.
  • JavaScript ten belli bir class a ulasmak icin " getElementByClassName() "
  • Class name case sensitive dir.

iframe ile ilgili bilgilere ulasmak icin lutfen tiklayiniz.
